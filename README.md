## 🚀 How to Use This Repository
Each project is contained within its own folder. Inside each folder, you will find:
1. A detailed `README.md` explaining the problem, methodology, and results.
2. A `notebook.ipynb` containing the full code, analysis, and visualizations.
3. A `requirements.txt` file listing the necessary dependencies.

To run a project locally:
```bash
git clone https://github.com/imSheheryar/Data-Science-Projects-Portfolio.git
cd Data-Science-Projects-Portfolio/<project-folder>
pip install -r requirements.txt
jupyter notebook notebook.ipynb
```
